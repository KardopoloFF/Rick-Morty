# Rick-Morty
Frontend App with Rick&amp;Morty API

Here you can:

- Find a hero by type numbers in SearchBar from 1 to 600+
- Add your hero to favorite
- Remove card from favorite 
- Delete all cards
- Watch full hero-card

___________________________

How to start

1. Clone repo
2. "npm i" 
3. npm start

- Here you go :)
